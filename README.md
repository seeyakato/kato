# kato
デザイン課題
======================
デザイン課題です  

使い方
------
このテンプレートをpushする手順の例です。

このリポジトリをクローンします。

    $ git clone https://github.com/seeyakato/kato.git

クローンしたリポジトリの中へ移動します。

    $ cd kato  
  
編集します  

`README.md` を新しいプロジェクトに合わせて編集します。

    $ $EDITOR README.md

コミットします。

    $ git commit -m "デザイン課題" 

プッシュします。

    $ git push
